# Market Cap Buckets

Valid values for the `--market-cap-bucket` / `marketCapBucket[]` parameter.

API value        | Approx range       | HTML label
---------------- | ------------------ | ----------------------------
Micro Caps       | $1 to $100m        | $1 to $100m
Small Caps       | $100m to $1b       | Greater than $100m to $1b
Mid Caps         | $1b to $10b        | Greater than $1b to $10b
Large Caps       | Over $10b          | Greater than $10b
Suspended        | Suspended companies | Suspended
Other            | No market cap data  | Other

Note: the HTML dropdown labels differ from the API parameter values.
Use the **API value** column when calling the API.
